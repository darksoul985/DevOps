#### Ответы к задаче 2

- высоконагруженная база данных MySql, критичная к отказу.
  Ответ: физические сервера.
  Пояснение: Физические сервера дают прямой доступ к ресурсам, исключая расходы на виртуализацию. За счет создания кластера можно увеличить отказоустойчивости, доступ к низкоуровневым настройкам непосредственно "железа" для оптимизации работы базы

- различные web-приложения.
  Ответ: паравиртуализация.
  Пояснение: Дешевле физических серверов. Легко создавать, клонировать, удалять ВМ. Гипервизор позволяет гибко распределять ресурсы.

- Windows-системы для использования бухгалтерским отделом.
  Ответ: паравиртуализация.
  Пояснение: Гипервизоры позволяет легко управлять рабочими столами, централизованное хранение данных, быстрое развертывание, гарантированная работа специализированного ПО.

- системы, выполняющие высокопроизводительные расчёты на GPU.
  Ответ: физические серверы.
  Пояснение: прямой доступ к GPU, максимальная производительность, поддержка CUDA/OpenCL, масштабируемость
